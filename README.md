# TypeScript VT2026

All källkod för YH-kursen _TypeScript_ för klasserna YKTYP26V1/YKTYS26V1 på IT-högskolan & Medieinstitutet.

## Källkod

Varje lektion är en egen, fristående branch och innehåller samtlig källkod från föregående branches.

## Branches

Hämta ut den branch du är intresserad av.

### Exempel

```zsh
git pull
git checkout lesson-1
cd 01-intro




npm init -y
npm -D typescript

npx tsc -v
npx tsc --init




git clone [http address ]
```
